# Unpacking-Workplace-Dynamics

### Project Overview
Analyzed workforce demographics using SQL and Power BI to evaluate the distribution of job roles across age and gender groups, uncovering patterns in role specialization, workforce aging, and employee retention. Built interactive dashboards to highlight succession risks, generational trends, and insights to support data-driven workforce and HR planning.

### Tools Used
- Power BI (DAX, Power Query,Data modelling)
- SQL
